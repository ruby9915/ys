소프트웨어 개발 실습

수업 목표

1. 웹 개발자가 되기 위해 필요한 Front-end의 전반적인 기초 지식 습득
2. 웹 개발을 위한 HTML5, CSS3, JavaScript의 기본 지식 습득
3. 개인별 웹페이지 제작 term-project를 통한 웹 페이지 개발 실전 경험

수업 개요

1. 웹과 Front-end의 전반적인 개념과 구조를 이해함
2. 웹 개발에 필요한 텍스트 에디터(VS code)의 사용 방법을 이해함
3. HTML5+CSS3의 개념을 습득하고 다양한 태그의 사용과 스타일시트의 적용 방법을 이해함
4. 오픈소스 언어인 JavaScript의 기본 문법 및 주요 API를 설명하고 웹 페이지 개발 실전 경험을 쌓음
5. 오픈소스 프로젝트 React의 기초 개념을 이해함

수업 맛보기

HTML

HTML(HyperText Markup Language)
HTML은 Hyper Text Markup Language 줄임말로 간단하게 말하면 웹 페이지를 만드는 마크업 언어라고 할 수 있다.

HyperText Markup Language

HyperText(웹페이지에서 다른 페이지로 이동할 수 있는) 기능을 가진 문서를 만드는 언어
문서를 구조적으로 Markup(표시하기)하여 웹 브라우저에게 명령하는 언어

CSS

CSS (Cascading Style Sheet)
웹 컨소시엄에서 웹 문서용으로 개발한 스타일 시트 언어라고 할 수 있다.
모듈 기반으로 개발하고 디바이스에 따라 원하는 모듈만을 탑재할 수 있음.

스타일 시트 & CSS

문서가 사용자에게 보여지는 방법을 지정하여 HTML 문서를 시각적으로 꾸미는 역할
스타일 시트를 이용하면 크기, 색상 등의 스타일을 일괄적용한다.

JavaScript

동적인 웹 개발을 위해 반드시 사용되는 프로그래밍 언어
웹 개발을 위해서는 무조건 JavaScript를 사용해야 함(대안이 없음..)

자바스크립트 역할

오늘날 HTML, CSS와 함께 웹을 구성하는 요소 중 하나다
HTML이 웹 페이지의 기본 구조를 담당하고, CSS가 디자인을 담당한다면
JavaScript는 클라이언트 단에서 웹 페이지가 동작하는 것을 담당한다.
